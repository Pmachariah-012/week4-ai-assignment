## 1. What is Data?
- Data refers to unprocessed raw facts or pieces of information that are collected and stored for analysis or used to make decisions. It includes; numbers, texts, images, audio, and videos.
  
*Examples:*

-The number of students in a class.

-Daily temperature readings in an area.

## 2. Difference between Structured and Unstructured Data
a) Structured Data
-This is data that is organized in a clear format such as rows and columns, making it easy to search and analyze using databases. Has a fixed schema.

*Examples:*

-A spreadsheet of employee salaries.

-A table of student records (name, age, grade).

b) Unstructured Data
-This is data that does not follow a fixed schema and is harder to organize in tables.

*Examples:*

-Images or photographs.
-Social media posts, audios or videos.

## 3. What is a Schema?
A schema is the structure or blueprint that defines how data is organized in a database. It shows the fields, data types, and relationships between pieces of data.

*Example*

A schema for student database table:

|   **Name**    |   **Gender**  |   **Age**    |   **Course**    |
|---------------|---------------|--------------|-----------------|
|   John        |  Male         |   24         |      AI         |
|   Mary        |  Female       |   22         |     Programming |


## Bonus question 

Differentiate between a list and a dictionary

- A list- Is a type of variable that can hold multiple pieces of data in order, like a row of items. In a list, each item can be accessed by its position.
A dictionary is also a variable that stores multiple pieces of data, but each item has a name or key instead of position. So with a dictionary, data is accessed by its name not by order.
